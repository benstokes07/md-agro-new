# **App Name**: AgriStatic

## Core Features:

- Homepage Display: Showcase key agriculture products and company information.
- About Us Page: Provide details about the company's mission, vision, and team.
- Product Catalog: List agriculture products fetched from JSON/Markdown files.
- Static Product Pages: Generate static pages for each product from JSON or Markdown data with permanent URLs.
- Contact Form: Allow users to submit inquiries via a contact form; consider a Netlify forms integration.

## Style Guidelines:

- Primary color: Earthy Green (#6B8E23) to reflect agriculture.
- Background color: Light Beige (#F5F5DC) for a natural, clean feel.
- Accent color: Harvest Gold (#DAA520) to highlight important information.
- Body font: 'PT Sans' a humanist sans-serif that combines a modern look and a little warmth or personality; good for readability.
- Headline font: 'Alegreya	' a humanist serif with an elegant, intellectual, contemporary feel.
- Use simple, recognizable icons related to agriculture products and benefits.
- Maintain a clean, corporate layout with clear sectioning and ample spacing for farmer-friendly readability.
